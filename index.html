OWASP GENAI v1.0

12 Menaces OWASP GenAI

Contrairement à OneTrust et Vanta qui ne couvrent que OWASP Top 10 traditionnel, Cortex Leman est le premier audit à couvrir OWASP GenAI Agentic AI (9-Phase SecOps) + OWASP GenAI Data Security v1.0 (12 menaces).

CRITICAL

Data Poisoning

Injection de données malveillantes dans training data

Mitigations:
  • Data sanitization pipeline
  • Outlier detection
  • Adversarial filtering
  • Immutable training data (WORM)
HIGH

Data Extraction

Extraction non autorisée de training data

Mitigations:
  • Differential privacy (DP-SGD, DP-SGD)
  • Output filtering
  • Response length limits
  • Rate limiting sur model queries
HIGH

Model Inversion

Reconstruction de training data depuis le modèle

Mitigations:
  • Gradient shielding during inference
  • Model regularization (L1/L2, dropout)
  • Output confidence thresholds
  • Shadow models for anomaly detection
CRITICAL

Backdoor Attacks

Insertion de triggers cachés dans le modèle

Mitigations:
  • Pruning fine-tuned layers before deployment
  • Outlier detection during inference
  • Adversarial testing before deployment
  • Backdoor detection (trigger scanning)
HIGH

Adversarial Examples

Input perturbés pour tromper le modèle

Mitigations:
  • Adversarial training (FGSM, PGD, CW)
  • Input perturbation detection
  • Ensemble voting
  • Input preprocessing (defenses)
HIGH

Context Extraction

Extraction de données sensibles depuis context LLM

Mitigations:
  • Context masking (redaction, anonymization)
  • Input/output filtering
  • Memory scoping (isolate sensitive context)
  • Context encryption
HIGH

Prompt Injection for Data

Injection de prompts pour extraire des données

Mitigations:
  • Input validation
  • Prompt filtering
  • Output sanitization
  • Prompt injection detection
  • LLM firewalls
