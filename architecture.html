Nouveaux Composants v5.2 NEW

RAG ChromaDB

  • Recherche sémantique vectorielle
  • Embeddings all-MiniLM-L6-v2
  • Isolation par client (RGPD droit à l'oubli)
  • Chunking 500 car. avec overlap
  • Context injection pour agents LLM
  • Textes réglementaires vectorisés
  • 5 endpoints: /rag/search, /rag/index, /rag/seed, /rag/stats, /rag/client/{id}

MCP Server

  • Model Context Protocol (JSON-RPC 2.0)
  • Expose RAG via protocole standard
  • Compatible Claude Desktop, Cursor, etc.
  • Tools: rag_search, rag_regulatory, rag_stats
  • Transport SSE/stdio
  • 1 endpoint: POST /mcp

A2A Adapter

  • Agent-to-Agent Protocol
  • Bridge NATS ↔ protocole A2A standard
  • Compatible AG2, CrewAI, agents tiers
  • Messages: discover, send_message, get_status
  • 1 endpoint: POST /a2a

AutoDefense Guardrails

  • Inspiré du paper AG2 AutoDefense
  • 3 validateurs indépendants en parallèle
  • Vote majoritaire → block/pass
  • Détection anti-jailbreak
  • Détection prompt injection
  • 1 endpoint: POST /guardrails/autodefense

Monitoring & Tracing

  • Prometheus metrics sur /metrics
  • Grafana dashboard préconfiguré
  • OpenTelemetry distributed tracing
  • Spans par agent (latence, health)
  • Dégradation gracieuse si OTel absent
  • 2 endpoints: GET /health, GET /metrics

Chiffrement Fernet

  • Chiffrement at-rest (Fernet/AES)
  • PII et secrets clients protégés
  • Conforme RGPD Art. 32
  • Rotation des clés supportée
  • Module: core/security/encryption.py
