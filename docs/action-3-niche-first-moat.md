## Stratégie: Niche-first, pas Acquisition-first

### ZénithIA (FR, volume)
```
Acquisition (300 mails/j) → Niche (toutes) → Vente → Projet ponctuel
```

### Cortex Leman (FR-CH, qualité)
```
Niche (RGPD-IA FR-CH) → Scoring (SocialPulse) → Audit → Conformité récurrente
```

### Avantages de notre approche
1. **Moins de leads, mieux qualifiés** — Un CFO romand ne répond jamais à un mail de masse
2. **Taux de conversion plus élevé** — Le prospect qui arrive sur "RGPD + IA + Suisse" est déjà qualifié
3. **Récurrent naturel** — La conformité n'est jamais "finie" → abonnement mensuel logique
4. **Références bouche-à-oreille** — Le marché romand est petit, la réputation se construit vite
5. **Barrière à l'entrée** — Un acteur FR ne peut pas copier notre expertise LPerD + secret pro CH

### Ce qu'on PEUT copier de ZénithIA
- ✅ Leur workflow de génération de propositions commerciales automatisées (Action 1)
- ✅ La technique "renvoyer les mots du client" (verbatim dans les rapports)
- ✅ Le format 3 tiers (Diagnostic → Audit → Conformité)
- ✅ Contenu → communauté → B2B comme entonnoir

### Ce qu'on NE copie PAS
- ❌ "Choisir l'acquisition avant la niche"
- ❌ 300 mails/jour en masse
- ❌ Vendre du temps (projet) au lieu de la conformité (récurrent)
- ❌ Ignorer le RGPD
